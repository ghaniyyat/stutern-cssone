Understanding HTML and CSS

HTML - Hyper Text Markup Language, is the language for documents that are designed to be displayed on a web browser. it is the language for describing the structure of web pages. It includes tags like html, head, title and body, and other element tags like headings (1-6), table, form etc.

CSS - Cascading Style Sheets, is the language used to style an HTML document. it describes how elements should be displayed on a web page.

There are three ways of adding CSS to an HTML document:
Inline CSS has the styling property written in the body section attached with element.
Internal CSS written in the head tag and is mostly used when to style a single html document (uniquely).
External CSS - this is written in a separate file with the .css extension and linked to the Html document using the link tag. It is mostly used to add styles across web pages.

Importance Of CSS

CSS is used for styling the html web page, and this basically is its importance.

By Styling, this covers a whole lot. It encompasses:
User Interface by adding colours designs, fonts and images to beautify a web page, adding images,
And sometimes User Experience by making the webpage look meaningful with the use of necessary layout and format, paddings, margins and even border to show the relativity/relation between contexts on web page, which without this, HTML appears clustered .

Therefore, the impotance of CSS on a web page is just as imperative as other languages used in building web pages.

Padding, Margin and Border

Padding is the space between a content and its border, it can be regarded as the inner spacing, and is usually between related elements .

Margin on the other hand, is the spacing around an element's border, thus can be regarded as the outer spacing.

While Border is the container wrapping an element or a content showing its visible outline.
